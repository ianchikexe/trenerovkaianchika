📆 ПРОГРАММА ТРЕНИРОВОК (8 НЕДЕЛЬ)

Каждую неделю — 6 тренировочных дней (пн–сб), воскресенье — отдых. Тренировка длится ~2 часа. Используй рюкзак с весом (5–15 кг) для отягощения. Таймер следит за отдыхом между подходами.

🍏 ПЛАН ПИТАНИЯ НА 2 МЕСЯЦА

🌅 08:00 ЗАВТРАК

Овсянка 80г + 3 яйца + авокадо + ягоды. Альтернатива: гречневые хлопья + омлет.

🕐 11:00 ПЕРЕКУС

Творог 200г + орехи 30г + горсть ягод. Протеиновый коктейль.

🍗 13:30 ОБЕД

Гречка/рис 120г + куриная грудка 180г + салат с оливковым маслом.

🥤 16:30 ПОЛДНИК

Греческий йогурт + банан + протеин + миндаль.

🐟 19:30 УЖИН

Рыба (форель/тунец) 180г + брокколи/кабачки + творог 100г.

🌙 22:30 ПЕРЕД СНОМ

Кефир + магний + казеин (творог). Никаких углеводов.

🚫 Полный запрет: хлеб, макароны, сахар, выпечка, фастфуд, сладкие напитки. Вода — 2.5 л/день.
