Шаг 1. Установите инструменты Перед началом убедитесь, что у вас установлен: Visual Studio (рекомендуется последняя версия) .NET SDK Совет: Если используете Linux/MacOS, установите Visual Studio Code, это тоже отличный вариант. Шаг 2. Создаем проект Запустите терминал и введите команды: dotnet new webapp --name MyFirstSite cd MyFirstSite dotnet run Проект успешно создан и запущен локально. Вы увидите стартовую страницу браузера. Шаг 3. Организация структуры файлов Наш проект теперь имеет стандартную структуру. Для начала давайте создадим раздел для нашего дизайна: Папка wwwroot: сюда мы будем помещать файлы HTML, CSS и JavaScript. Файлы HTML будут храниться отдельно, а наш движок будет отвечать за взаимодействие с ними. 💡 Пример структуры: wwwroot/ ├── css/ │ └── styles.css ├── js/ │ └── main.js └── pages/ └── index.html Шаг 4.
