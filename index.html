📆 ПРОГРАММА (8 НЕДЕЛЬ · 6 ДНЕЙ)

Каждый день 8–9 упражнений. Общее время ~2 часа. Отдых между подходами строго по таймеру. Нагрузка растёт каждые 2 недели. Воскресенье — отдых.

🍏 КРИВАЯ ТАБЛИЦА ПИТАНИЯ

🌅 08:00 ЗАВТРАК

Овсянка 80г + 3 яйца + авокадо + чиа. Никакого хлеба.

🕐 11:00 ПЕРЕКУС

Творог 200г + орехи 30г + черника. Протеин.

🍗 13:30 ОБЕД

Гречка 120г + куриная грудка 180г + салат (помидоры, огурцы, масло).

🥤 16:30 ПОЛДНИК

Греческий йогурт + банан + протеин + миндаль.

🐟 19:30 УЖИН

Форель/тунец 180г + кабачки/брокколи + творог 100г.

🌙 22:30 ПЕРЕД СНОМ

Кефир + магний + казеин. Никаких углеводов.

🚫 ПОЛНЫЙ ЗАПРЕТ: хлеб, макароны, сахар, выпечка, фастфуд, сладкие напитки. ВОДА 2.5 л/день.
